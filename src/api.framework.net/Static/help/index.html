This framework is designed for reducing the development efforts for web APIs. The framework has following inbuilt features, that can be controlled only by configurations.
  • Develop the GET endpoints with just writing a JSON file in a predefined format.
  • Automatic swagger documentation will be generated on the fly.
  • Detailed logging and tracing with request and trace ids.
  • It provide a health check URL, which can be used for health probe in load balancers.
  • Capability to integrate your API endpoint with SQL as backend by just simple configuration.
  • Capability of writing APIs with multiple backend operations.
  • Capability to write almost all types of input validations with configuration.
  • Capability to add custom validation by writing a code function and integrating into API with configuration.
  • Capability to add a coded backend where you can write any code and integrate with a API endpoint.
  • Capability to mock the SQL Backend, so that API will work with dummy mock data.
